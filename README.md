# VueScan

VueScan 是一个全流程自动化的 Vue 站点资产分析与 API 探测工具。通过智能化的路由收集、代码映射和参数模糊测试，帮助安全研究人员和开发者快速理解应用架构。已完整打通从路由访问、请求捕获、JS/API 提取，到 `BaseRequest` / `ApiRequest` 调试和自动化快照分析的端到端流程。

## 核心模块

### 📱 VueDetect 模块 - Vue 应用探测

快速识别和分类 Vue 站点资产：

- 支持从多种格式导入项目资产（`xlsx / txt / html` 等）
- 批量识别 Vue 站点并通过路由数量自动排序
- 快速定位大规模资产中的 Vue 框架应用

### 🔍 VueChunk 模块 - 核心代码分析与自动化探测

完整的资源映射和 API 探测工具链：

#### 项目与资源管理

- 创建和管理分析项目
- 访问路由并自动捕获 `route -> api -> chunk js` 的完整链路
- 智能下载 JS 资源到本地

#### API 分析功能

- **预览与提取** - 从 JS 代码中预览和批量提取 API 路径
- **代码定位** - 通过 `Locate JS` 精确定位 API 对应的源代码位置
- **智能正则生成** - 基于 LLM 自动生成正则表达式，批量收集 API
- **基址推断** - 自动推断 `BaseURL / BaseAPI` 配置

#### 请求构造与调试

- **BaseRequest 配置** - 灵活构造 `BaseURL / BaseAPI / BaseQuery / BaseHeader / BaseBody`
- **参数模糊测试** - 使用 `FuzzParams` 生成 `Query / JSON Body / Form Body` 等多种参数形式
- **ApiRequest 执行** - 基于配置的 BaseRequest 批量发送请求

#### 结果分析

- 自动执行和保存运行快照
- 支持 `GET / POST` 两轮结果对比分析
- 项目价值分析报告生成

## 快速开始

### 环境要求

- Python 3.8+
- Node.js 14+
- Chromium 浏览器

### 安装与配置

1. 安装 Python 依赖：

```bash
pip install -r requirements.txt
python -m playwright install chromium
```

2. 启动后端服务：

```bash
python vs.py
```

后端默认运行在 `http://127.0.0.1:8000`

3. 构建前端（可选，开发环境可使用前端开发服务器）：

```bash
cd frontend
npm run build
```

4. 访问应用：`http://127.0.0.1:8000/#/vueDetect`

### 配置 LLM 接口

在应用的全局设置中配置 LLM API 接口，支持多个大模型服务商（如 OpenAI、DeepSeek 等），用于自动正则生成和代码分析。

## 用户认证

系统已启用完整的登录鉴权机制。

### 首次初始化

首次启动应用时，系统需要初始化管理员账户：

1. 打开应用首页
2. 进入”初始化管理员”界面
3. 输入管理员账号和密码
4. 初始化成功后自动登录

### 登录与权限

- 所有业务接口均需要有效的登录状态
- 使用初始化的管理员账号登录即可访问全部功能
- 修改密码可在页面右上角齿轮菜单中进行

## 自动化工作流

启用自动化项目创建后，系统将执行完整的端到端分析流程：

### 自动化分析步骤

| 步骤  | 功能描述                                   |
| --- | -------------------------------------- |
| 1   | 访问并归一化目标 `route_url`                   |
| 2   | 拦截并捕获该路由下的所有 API 请求和 Chunk JS 资源       |
| 3   | 执行 `Locate JS` 操作，定位 API 在源代码中的位置      |
| 4   | 过滤结果：仅保留命中且 `js_api_path` 包含有效关键词的 API |
| 5   | 使用 LLM 自动生成正则表达式                       |
| 6   | 批量提取 API 端点和参数                         |
| 7   | 自动推断 `BaseURL` 和 `BaseAPI` 配置          |
| 8   | 智能构造 `GET / POST` 两种方式的 BaseRequest    |
| 9   | 自动执行双轮 ApiRequest 请求，对比验证              |
| 10  | 保存执行快照，生成项目价值分析报告                      |

### 关键特性

- ⚡ **全自动** - 从探测到分析全流程无需人工干预
- 🎯 **精确定位** - API 到源代码的精确映射
- 🤖 **LLM 赋能** - 智能正则生成和参数识别
- 📊 **完整对比** - GET/POST 双轮结果验证
- 💾 **快照保存** - 完整的执行记录和分析报告

## 使用场景

- 🔐 **前端安全审计** - 快速发现所有 API 端点及参数

- 🧪 **自动化测试** - 生成标准化的 API 探测用例
- 📈 **代码理解** - 精确定位功能实现的源代码位置
